# NodeConfigClient
Node config client to connect with config server e.g. spring config server.

## Installing
- Install yarn 1.5.1 or above. Visit https://yarnpkg.com/lang/en/docs/install/
- yarn install

## Examples
- See EXAMPLES.md file.

## Running the tests
* yarn test
